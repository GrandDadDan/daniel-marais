<h1>Tic Tac Toe 🎮</h1>
A classic Tic Tac Toe game built using React, Node.js, and Firebase. Play against a friend and enjoy a smooth, interactive experience!

<h2>Features</h2>
✔️ Two-player mode
✔️ Simple and intuitive UI
✔️ Real-time updates with Firebase
